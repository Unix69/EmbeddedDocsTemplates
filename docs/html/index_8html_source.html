160  <h2><img src="Images/Logo/doxygen.svg" alt="Doxygen" width="24px" height="24px"> Doxygen Customization</h2>
161  <b>README Template</b> use doxygen customization as follow:
162  <br>
163  <ul style="list-style-type: none; padding-left: 0;">
164  <li>‚úÖ HTML output enabled (<code>GENERATE_HTML=YES</code>)</li>
165  <li>‚úÖ Tree view in HTML enabled (<code>GENERATE_TREEVIEW=YES</code>)</li>
166  <li>‚úÖ Search engine enabled (<code>SEARCHENGINE=YES</code>)</li>
167  <li>‚úÖ Markdown supported as main page and in input (<code>MARKDOWN_SUPPORT=YES</code>)</li>
168  <li>‚úÖ Graphs enabled (Class, Collaboration, Include, Caller/Call Graphs, UML, Directory, Group)
169  (<code>CLASS_GRAPH, COLLABORATION_GRAPH, INCLUDE_GRAPH, CALL_GRAPH, CALLER_GRAPH, UML_LOOK, GROUP_GRAPHS, DIRECTORY_GRAPH=YES</code>)</li>
170  <li>‚úÖ Graphviz interactive SVG with multi-targets, transparent background, and cleanup (<code>INTERACTIVE_SVG, DOT_MULTI_TARGETS, DOT_TRANSPARENT, DOT_CLEANUP=YES</code>)</li>
171  <li>‚úÖ Warnings logged to <code>docs/doxygen_warnings.log</code> (<code>WARN_LOGFILE</code>)</li>
172  <li>‚úÖ Source browser and inline sources enabled (<code>SOURCE_BROWSER=YES, INLINE_SOURCES=YES</code>)</li>
173  <li>‚úÖ Extract all entities including private, static, local classes/members, packages, local methods, anonymous namespaces (<code>EXTRACT_ALL, EXTRACT_PRIVATE, EXTRACT_STATIC, EXTRACT_LOCAL_CLASSES, EXTRACT_PACKAGE, EXTRACT_LOCAL_METHODS, EXTRACT_ANON_NSPACES=YES</code>)</li>
174  <li>‚úÖ Undocumented classes/members are shown (<code>HIDE_UNDOC_CLASSES=NO, HIDE_UNDOC_MEMBERS=NO</code>)</li>
175  <li>‚úÖ Sorting & organization: sorted by member, brief, and scope, section labels, index generated (<code>SORT_MEMBER_DOCS, SORT_BRIEF_DOCS, SORT_BY_SCOPE_NAME, ENABLE_SECTION_LABELS, GENERATE_INDEX=YES</code>)</li>
176  <li>‚úÖ HTML customization: header, footer, extra stylesheets (<code>HTML_HEADER, HTML_FOOTER, HTML_STYLESHEET, HTML_EXTRA_STYLESHEET</code>)</li>
177  <li>‚úÖ MathJax support for formulas (<code>USE_MATHJAX=YES, FORMULA_FONTSIZE=12</code>)</li>
178  <li>‚úÖ Autolinking support (<code>AUTOLINK_SUPPORT=YES</code>)</li>
179  <li>‚úÖ TOC includes headings up to level 3 (<code>TOC_INCLUDE_HEADINGS=3</code>)</li>
180  <li>‚úÖ Manual pages generation enabled (<code>GENERATE_MAN=YES</code>)</li>
181  <li>‚úÖ Enumerations displayed one value per line (<code>ENUM_VALUES_PER_LINE=1</code>)</li>
182  <li>‚úÖ Example files included recursively (<code>EXAMPLE_PATH, EXAMPLE_RECURSIVE=YES</code>)</li>
183  <li>‚úÖ References and relations shown (<code>REFERENCED_BY_RELATION=YES, REFERENCES_RELATION=YES</code>)</li>
184  <li>‚úÖ Verbatim headers displayed (<code>VERBATIM_HEADERS=YES</code>)</li>
185 </ul>

193  <h2><img src="Images/Logo/doxygen.svg" alt="Doxygen" width="24px" height="24px"> Doxygen Code Aliases</h2>
194  <p>In your source code, you can use these custom aliases in comments. Doxygen will render them with icons and labels.</p>
195 
196  <ul style="list-style-type: none; padding-left: 0; font-size: 18px; line-height: 2;">
197  <li>üõ†Ô∏è <b>@fix</b> ‚Äì Use for code fixes or patches.</li>
198  <li>üêû <b>@bug</b> ‚Äì Use to document a bug found or fixed.</li>
199  <li>üîÑ <b>@change</b> ‚Äì Use to describe a change in functionality or logic.</li>
200  <li>üìù <b>@todo</b> ‚Äì Use to mark something that needs to be done.</li>
201  <li>üîó <b>@api</b> ‚Äì Use to highlight API-related notes.</li>
202  <li>üì¶ <b>@namespace</b> ‚Äì Use to describe a namespace.</li>
203  <li>‚ú® <b>@feature</b> ‚Äì Use to document a new feature.</li>
204  </ul>
