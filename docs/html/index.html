Doxygen Code Aliases

In your source code, you can use these custom aliases in comments. Doxygen will render them with icons and labels.

  • üõ†Ô∏è @fix ‚Äì Use for code fixes or patches.
  • üêû @bug ‚Äì Use to document a bug found or fixed.
  • üîÑ @change ‚Äì Use to describe a change in functionality or logic.
  • üìù @todo ‚Äì Use to mark something that needs to be done.
  • üîó @api ‚Äì Use to highlight API-related notes.
  • üì¶ @namespace ‚Äì Use to describe a namespace.

Doxygen Customization

README Template use doxygen customization as follow:
  • ‚úÖ HTML output enabled (GENERATE_HTML=YES)
  • ‚úÖ Tree view in HTML enabled (GENERATE_TREEVIEW=YES)
  • ‚úÖ Search engine enabled (SEARCHENGINE=YES)
  • ‚úÖ Markdown supported as main page and in input (MARKDOWN_SUPPORT=YES)
  • ‚úÖ Graphs enabled (Class, Collaboration, Include, Caller/Call Graphs, UML, Directory, Group) (CLASS_GRAPH, COLLABORATION_GRAPH, INCLUDE_GRAPH, CALL_GRAPH, CALLER_GRAPH, UML_LOOK, GROUP_GRAPHS, DIRECTORY_GRAPH=YES)
  • ‚úÖ Graphviz interactive SVG with multi-targets, transparent background, and cleanup (INTERACTIVE_SVG, DOT_MULTI_TARGETS, DOT_TRANSPARENT, DOT_CLEANUP=YES)
  • ‚úÖ Warnings logged to docs/doxygen_warnings.log (WARN_LOGFILE)
  • ‚úÖ Source browser and inline sources enabled (SOURCE_BROWSER=YES, INLINE_SOURCES=YES)
  • ‚úÖ Extract all entities including private, static, local classes/members, packages, local methods, anonymous namespaces (EXTRACT_ALL, EXTRACT_PRIVATE, EXTRACT_STATIC, EXTRACT_LOCAL_CLASSES, EXTRACT_PACKAGE, EXTRACT_LOCAL_METHODS, EXTRACT_ANON_NSPACES=YES)
  • ‚úÖ Undocumented classes/members are shown (HIDE_UNDOC_CLASSES=NO, HIDE_UNDOC_MEMBERS=NO)
  • ‚úÖ Sorting & organization: sorted by member, brief, and scope, section labels, index generated (SORT_MEMBER_DOCS, SORT_BRIEF_DOCS, SORT_BY_SCOPE_NAME, ENABLE_SECTION_LABELS, GENERATE_INDEX=YES)
  • ‚úÖ HTML customization: header, footer, extra stylesheets (HTML_HEADER, HTML_FOOTER, HTML_STYLESHEET, HTML_EXTRA_STYLESHEET)
  • ‚úÖ MathJax support for formulas (USE_MATHJAX=YES, FORMULA_FONTSIZE=12)
  • ‚úÖ Autolinking support (AUTOLINK_SUPPORT=YES)
  • ‚úÖ TOC includes headings up to level 3 (TOC_INCLUDE_HEADINGS=3)
  • ‚úÖ Manual pages generation enabled (GENERATE_MAN=YES)
  • ‚úÖ Enumerations displayed one value per line (ENUM_VALUES_PER_LINE=1)
  • ‚úÖ Example files included recursively (EXAMPLE_PATH, EXAMPLE_RECURSIVE=YES)
  • ‚úÖ References and relations shown (REFERENCED_BY_RELATION=YES, REFERENCES_RELATION=YES)
  • ‚úÖ Verbatim headers displayed (VERBATIM_HEADERS=YES)
