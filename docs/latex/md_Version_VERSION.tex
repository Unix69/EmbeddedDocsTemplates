\href{\#table-of-contents}{\texttt{ Table of Contents}}

Every release of $\ast$$\ast${\bfseries{Readme Template}}$\ast$$\ast$ follows a standard lifecyle that is identified by Semantic Versioning Standrard into the following phases\+:


\begin{DoxyItemize}
\item {\bfseries{Pre-\/\+Release}}\+: release is unstable and it is not completely ready to be used, some changes should be done or maybe some bugs occur so fixes should be performed.
\begin{DoxyItemize}
\item {\bfseries{Beta}}\+: release is completely unstable and features are not complete and correctly implemented, some important changes should be done or maybe some bugs occur so fixes should be performed.
\item {\bfseries{Alpha}}\+: release is stable and all features are complete and correctly implemented, maybe some bugs occur and fixes should be performed.
\end{DoxyItemize}
\item {\bfseries{Release}}\+: release is stable and and all features are complete and correctly implemented, no official bugs occur.
\item {\bfseries{Obsolete}} release is still supported but will be abbandoned at the next version.
\item {\bfseries{Deprecated}} release is not anymore supported and became unstable.
\end{DoxyItemize}
