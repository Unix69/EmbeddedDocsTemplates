\href{\#table-of-contents}{\texttt{ Table of Contents}}

The R\+E\+A\+D\+ME Template is designed to support\+:


\begin{DoxyItemize}
\item Multi-\/level documentation (User, Admin, Developer)
\item Version tracking under {\ttfamily /\+Version}
\item Feature/\+A\+P\+I/\+Namespace documentation
\item Automatic rendering using Doxygen
\item H\+T\+ML + Markdown synchronization
\item Modular architecture
\end{DoxyItemize}

The development workflow involves modifying\+:


\begin{DoxyItemize}
\item Markdown content ({\ttfamily /\+Usage}, {\ttfamily /\+Version})
\item Doxygen config ({\ttfamily Doxyfile})
\item Template assets ({\ttfamily \mbox{\hyperlink{header_8html}{header.\+html}}}, {\ttfamily \mbox{\hyperlink{footer_8html}{footer.\+html}}}, {\ttfamily \mbox{\hyperlink{stylesheet_8css}{stylesheet.\+css}}})
\item Build scripts ({\ttfamily Makefile})
\end{DoxyItemize}

All contributions should preserve template coherence and consistent linking across documents.

To work on the template structure, set up your environment as follows\+:


\begin{DoxyEnumerate}

\item Ensure you have the required tools\+: ~\newline

\begin{DoxyItemize}
\item Doxygen ~\newline

\item Graphviz (optional, for diagrams) ~\newline

\item G\+NU Make ~\newline

\end{DoxyItemize}
\item Build the documentation locally\+: ~\newline
 {\ttfamily make build} ~\newline
 {\ttfamily make doc}
\item Open\+: ~\newline
 {\ttfamily docs/html/index.\+html}
\end{DoxyEnumerate}

This environment allows you to test changes to Markdown files, navigation links, Doxygen tags, and H\+T\+ML templates.
